IDVN#2020-101000001

Admin Panel Indihome pada perangkat huawei DP-05822 rentan akan serangan cross-site request forgery pada menu admin [14 Oktober 2020, 13:00]

Overview CSRF menggunakan script inject pada halaman login admin panel untuk melakukan bypass user login.

Products Affected Huaweu DP-0598 Indihome.

Impacts Penyerang dapat mengambil alih sistem.

Description Serangan dapat dilancarkan dengan menggunakan halaman admin panel dengan default url 192.168.100.1 dan inject CSRF dengan menggunakan script CSRF Injector Indihome.

Workaround Nonaktifkan sementara halaman admin panel, set default akses menggunakan kabel LAN only.
